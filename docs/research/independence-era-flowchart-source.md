# Independence-Era Event Flowchart — Source Extraction (v2, per-node)

> **Provenance:** machine-extracted from the user's Canva "Era Evos Flow Chart" (`DAGLnKhUNtk`, Joshua Sweetman), the design contract for the phase 2.4.3 branching event-graph expansion.

> **Coverage:** TEXT is essentially complete — 1894/1896 unique strings on v1; this v2 adds the 2 stragglers and is extracted per-text-box (each node's lines joined) with its fill colour.

> **CRITICAL LIMITATION:** this file has all node *text* but NOT the graph *edges* (which node leads to which, spine vs branch). **Connectivity lives in the two screenshots** (`IMG_1889.png`, `IMG_1890.png`) — they are authoritative for structure. Nodes below are in Canva document order, which ≈ the author's narrative order (Independence first, then Federalism), but cross-check the screenshots before trusting any edge.

> **Era boundary:** Independence ≈ nodes 1–~40 (Gaspee Affair → Constitutional Convention / Federalist Papers + the Iroquois/Cherokee treaties). The **French Revolution** node begins the Federalism economic arc (Buttonwood, Compromise of 1790, Whiskey/Excise tax, Louisiana, War of 1812, Napoleon…) — that is the NEXT era's build.
