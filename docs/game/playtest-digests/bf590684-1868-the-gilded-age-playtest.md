## What this thread is

The **first dedicated, native 1868-start Gilded-Age multiplayer campaign** in the
KB (every prior Gilded coverage — `gilded` f4c7c2c4, `house-divided`, `drums`,
`tea1772` — *reached* 1868 as a continuation of an earlier start). It boots
mid-government in 1868 (post-Civil-War one-party GOP dominance: Senate 56-9 Red,
House 141-70 Red, POST 71), runs the **full Gilded-Age arc 1868 → ~1886** over
many election/convention cycles, and **ends — like `new1772` — in GM burnout**
(a 1884-86 GM/player conflict over a Lifetime-President amendment, POST 6273-6318).
It **never reaches Progressivism** (KB negative result holds: no thread reaches a
"future" era). It is the **richest single record of the Gilded-Age issue engine
actually being played** — tariff %, currency, civil-service, trusts, the 20-year
Reconstruction regime, African-American enfranchisement, imperialism — and
corroborates the existing Gilded cluster (#3/#5/#6/#21/#41) from a native start
while surfacing several NEW Gilded-specific gaps.

## Mechanics exercised (bucketed; shipped-vs-designed)

This thread is mostly the **unbuilt design** (the Gilded Age runs on `modern`
tuning with no scenario — gap #41), so nearly every mechanic below is
*designed-not-built*. It exercises the **entire phase taxonomy** (2.1.x draft/
career/relocate/ideology/kingmaker → 2.2.x leadership → 2.3.1 cabinet → 2.4.x
death/anytime/era events → 2.5.x lingering/gov-actions/SCOTUS → 2.6.x legislation
→ 2.8.x exec actions → 2.9.x convention/election) end-to-end across ~9 cycles.

- **Draft / career / development** (POST 98, 145, 193): 30%-off-profile draft;
  career-track spec verbatim (auto-removed @20yr, ideology→interest table,
  Overeager/Late-Bloomer); ideology-shift ±1 step **"cannot move LWPop↔RWPop"**
  (in 1868 the scale is LINEAR — the `rep1800` "ideology is a circle" rule
  #99 is NOT active here); kingmaker/protégé spec verbatim (POST 193).

  Min/Maj-Leader offices uncreated (~1899 historically — offices-by-law);
  Faction-Leader 6-criterion filter + **explicit voiding order**; Party-Leader
  IRV + on-elect bundle (+1 cmd, lose obscure, +3 primaries, ideology shifts,
  Charisma→+1 party-pref meter).

## GA rulings & house rules

- **3.0.17 "1872 Rule" / Special Election Conditions** (Tyler, POST 49, 774):
  the disorganized post-CW losing party (the South) runs **opposite-party
  independents** as its Pres/VP nominee — *gated by meters*: first election after
  Reconstruction begins, if party-pref Red+2/+3 AND a d6 roll of 1-2, the GM
  picks an independent-GOP ticket and has the lowest-scoring Dem "run" it.
- **20-year auto-expiring Reconstruction** (OrangeP47/GM, POST 73, 76): begins
  1864, **auto-ends 1884** ("there to prevent a one-party state"); during it,
  southern Gov/Sen/Rep seats are **appointed not elected** (military Govs Pres-
  appointed but majority-Congress-party; Senators by PPT-faction, Reps by
  Speaker-faction; appointees can't have seceded — rules 3.0.32/3.0.35); +2 GOP
  appointment/election bias; sunsets to **Blue+5 Solid South** (POST 5145).
- **Secessionist data tag** (Tyler/Umbrella, POST 167, 178): no real Secessionist
  trait — hand-tagged a **1/0 "union loyalist" column** (1=loyalist, 0=seceded,
  blank=non-Southern); seceded pols inactive until pardon. Wish: a real
  Secessionist trait (POST 165). vcczar deferred on the rule (POST 418).
- **Convention 1st-ballot ¼-delegate allocation** adjudicated by Tyler as "guru
  of conventions" (POST 3677): office-control → state-ideology-pref → momentum →
  party-leader → random.
- **15th Amendment effect** (POST 434): "Incumbent party +2 next election only;
  African-American men can hold office" — AfAm enfranchisement as a scored
  amendment; minority rookies who **haven't earned the vote can't be career-
  tracked** (POST 145, 152).
- **Dataset hygiene** (vcczar, POST 134): "remove everyone whose death date
  arrived before 1868" (Robert Gould Shaw 1864 draft date after 1863 death).
- **ebrk/Ted clarifications:** failed anytime-event roll still grants +1 midterm
  (POST 691); cabinet bonuses applied AFTER the confirmation vote (POST 610);
  PPT serves on committee only if majority >60% or minority would otherwise hold
  a committee majority (POST 447).
- **Convention-nominee must quit office; non-nominees can still run** (GM, ch55
  POST 829).

## Bugs / dataset / friction

- **Currency exclusivity bug** (POST 6245): Bimetallism + Gold Standard both
  active despite being mutually exclusive — no bill-relationship/exclusivity
  guard.
- **Dataset hygiene**: pol draft-date > death-date (R.G. Shaw, POST 117); a
  3-year-old "running around in 1840" (POST 124); NM rep seat existing pre-1912
  statehood (hidden-row hack, POST 374); Josiah Tattnall wrong bio (POST 169);
  governor term-lengths not historically accurate (NY, POST 141) — all hand-
  fixed by GM.
- **No Secessionist trait** in the dataset — built by hand (POST 167).
- **Gov-action meter-move is rare** (pass-roll THEN separate ~5% meter roll, POST
  803) → DomStab crisis nearly unsolvable at the state level; players spam
  "Increase State Gov Jobs" fruitlessly (POST 805).
- **Filibuster carry-over rule unclear** (POST 939) — matches the `drums`/`hd`
  open question.
- **GM friction**: a first-time GM reading rules live caused many re-do's
  (kingmaker rewards POST 250, cabinet-bonus timing POST 588, amendment 1-vs-2-
  turn ambiguity POST 437-445 [rules "say both depending where you look"]).
- **Thread ends in GM/player conflict** (POST 6273-6318) over the Lifetime-
  President amendment + time-off disputes — the campaign dies ~1886.
