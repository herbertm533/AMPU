# Digest — "The Big Red Button: 1960 Playtest" (`8bc0231c`)

- **Source:** `docs/game/sources/8bc0231c-ampu-the-big-red-button-1960-playtest/` — 1727 posts / 23 chunks (chunk-001…023).
- **GM / GA:** **@Cal** (game admin; runs the early phases), with **@Arkansas Progressive** (AP) taking over rolling from ~the general election onward. **NOT designer-authoritative** by default — but **@vcczar** (designer) and **@MrPotatoTed** (Ted, designer-class) are *active players in this run* and make in-thread rulings + design decisions that ARE authoritative (Ted plays the Corporatists/JFK; vcczar weighs in on platform/election tuning). Cite designer posts as authoritative; Cal/AP procedure as house-call.
- **Format:** 10-human multiplayer, modern polarity **BLUE = Democrats / RED = Republicans** — but **pre-realignment** (the South is still BLUE; the Democratic big tent holds both Thurmond's segregationist Southern Democrats AND Humphrey/Kefauver civil-rights liberals — see `historical-context-1960-coldwar.md`). Never reaches the 1964 presidential cycle. Original players decamped to the 1916 test; AP attempted a solo resume that petered out (POST 1726).
- **Why it matters:** The **FIRST Cold War / Civil Rights-era source in the corpus**, and an explicit **modern-era stress test at larger scale** than the early-American playtests (GM, POST 1). Exercises presidential primaries→brokered convention→general at full 50-state scale, the platform→ideology-meter engine, modern legislation/cabinet/SCOTUS/gov-actions/lingering-phase machinery, and Cold-War event/legislation content — all hand-adjudicated.

## Headline divergences (alt-history; do NOT "correct" to reality)
See `historical-context-1960-coldwar.md` §"Divergences" for the full treatment. In brief:
1. **GOP nominated John W. Bricker, not Nixon** (Nixon declined to run, POST 58; MacArthur staged a convention walkout that left Bricker the only option, POST 335).
2. **Bricker WON the 1960 presidency** → a **President Bricker alt-timeline** governs 1961-64. (See "Election-formula bug" below for the disputed margin.)
3. **President Bricker signs an alt Nuclear Test Ban Treaty** in the 1962-64 scripted block (POST 1677-1678).

5. **MLK scripted to be drafted into office in 1964** (POST 1677).
6. The **"big red button" nuclear-war loss condition is premise/flavor only — it NEVER fired** (the run paused before any Cuban-Missile-Crisis analog).

The single most build-relevant cluster. The 1960 scenario was **not authored as an Elections-Phase start** — players forced one and the GM patched holes live:

## 2. Presidential primaries → brokered convention (largest convention capture in KB; corroborates #19/#20/#181)
The fullest modern primary+convention trace in the corpus. Rules surfaced:

**Primary structure (1960 = transitional era):**
- **12 primary states**; most delegates won at convention. To run for president, a statesman needs **≥1 Command** (POST 4).
- Each faction may run **1 Major + 1 Minor candidate.** Auto-Majors: incumbent president, faction leaders, and "celebrity"-trait holders. **Minor = favorite-son** (only wins his home state) but can emerge as a Major if the convention deadlocks (POST 4).
- **Primaries run in Groups** (NH → WI/IL → NJ/MA → PA/DC/IN/OH → Group 5 cluster). Per group: GM determines frontrunner (must be a Major) → **debate** (if the party has one) → **scandal rolls** → **Major-candidate primary actions** → roll results (POST 73).
- **Per-group actions** (each candidate may take **several**): *Campaign Focus* on a state (±1 rally roll, can backfire), *Major/Policy Speech* (Orator gives +2 to all primaries in the group), *Promise VP*, *withdraw/endorse*. Debate winner +1 next group / loser −1 (POST 108, 135, 175).
- **Delegate weights** (the masterstroke lever): one governor per party assigns each state to Cat 1-5 (1×/2×/3×/4× EV; Cat 5 = bias-conditional). **A player gerrymandered the GOP weights so RW-Populist MacArthur's strongholds had few delegates and liberal Percy's had many** — decisive (POST 79, 172, 352#3).
- **Kingmakers do NOT affect primaries** (only conventions); Kingmaker→Protégé chains give +1 (POST 117-118).

**Convention / interballot:**
- **67% needed on first ballot** (changeable). **Two faction leaders can force a rule-change vote**; every faction votes with its **Kingmakers** (straight majority to change). Possible changes: lower threshold to 50%+1, winner-take-all, proportional (POST 4).
- **Interballot turn order**: GM imposed **least-delegates-first** order (so powerhouses move last) and required a fixed schedule because move-order is strategically load-bearing (POST 329).
- **The 100%-delegate rule** was voted in to block MacArthur — then MacArthur **walked out**, collapsing the field and leaving **Bricker the only nominee** (POST 352-353, 335). 100% remained an active rule into 1964 (POST 355).
- **VP nomination:** decided by votes weighted by **Kingmaker count**; party leader + nominee each name a VP; a **presidential promise** overrides the vote; **VP can't share the nominee's home state**; tie → party leader picks from lowest-scoring faction (POST 335).
- **Ticket-composition meter effects:** same-faction VP → that ideology enthusiasm +2 / everyone else −1; cross-faction → pres+VP ideologies +1 / others −1 (POST 371-372). Bricker/Laxalt (both Conservative, same faction) and JFK/Reagan (both Controversial+charismatic) each produced documented enthusiasm swings (POST 365).

**Design-hole flagged by players (NEW, #183):** **withdrawal/endorsement has NO mechanical effect unless the dropper holds actual delegates** (only ideological-split / home-state delegate reallocation matters). pman: a real-world endorsement should carry momentum (Buttigieg→Biden 2020) regardless of delegates. Proposals: +2 minor / +3 major endorsement, or endorsee inherits the endorser's home-state/region bonus or ideology-penalty waiver. Counter: d6 base leaves little room (POST 240-249).

## 3. The platform engine is OVERPOWERED — designer-confirmed nerf (★ corroborates #19/#20/#181/E-cluster)
The biggest single mechanical finding for elections.

- **Platform mechanic:** the winning ticket's nominee picks **4 platform planks** (Domestic / Finance-Economic / Foreign / Judicial) + an executive action; each plank moves **ideology-enthusiasm meters** ±1/±2 toward/away from a party. If pres is **Pliable** and party leader is **Manipulative**, the **party leader picks all planks** (POST 337-345). Platform impacts **party preference + ideology meters** but (per players) "that's all that happens" at the convention — no roll for the VP reveal, no candidate-quality gate (POST 581).
- **★ Result:** Bricker's clever conservative platform produced a **492-40 Bricker landslide over JFK** (POST 532) despite JFK running "a flawless campaign" and Bricker being a non-incumbent ex-senator. Players + designers unanimously: **platform is way too powerful** (Ted+vcczar: "Agreed — platform is way too powerful," POST 538-540). vcczar: "Ok I'll decrease the power when I get to that part of the rules" (POST 540).
- **★ Election-formula bug found:** Bricker was mistakenly given **+3 in every state** for the rule *"+1 to candidate with most enthusiastic personal base (unless >2 ideologies from state pref)"* — a **+1 max** misread as +3 (POST 603). Recomputes:
  - Remove the bogus **+3 platform bonus** → **Bricker 284 – Kennedy 251** (POST 587, 596). *(NB: the historian doc lists "284-251" as the *initial* tally; in fact the canonical original was 492-40 and 284-251 is the *first recompute* with the +3 removed.)*
  - Correct the per-state +3 → +1 → **Bricker 281 – Kennedy 129 – Tied 130** (CA & NY flip to ties) → a near-tied, JFK-slight-favorite map (POST 605-607).
- **★ GM kept the original 492-40 as canon** — "Bricker and his team earned their landslide victory and followed the rules that existed at the time, not going to take that from them. Just adjusting how it goes in the future" (POST 601). So the alt-timeline stands on the *buggy* result.
- **Tuning decisions taken (designer-blessed direction, not yet shipped):** cap platform meter-moves to **+2/+1/−1** (highest/second/lowest); change end-of-election roll **d6 → d3**; raise historical-bias from **+1/+2 → +3/+5** (so outcomes lean less on raw luck) (POST 598-600, 607). Alternatives debated: remove platforms entirely; make platform impact enthusiasm only *after* the promise is delivered; each faction submits exactly one plank (anti-collusion); plank limited to faction's own ideology (POST 564-582).
- **Down-ballot coattails:** the platform-driven Bricker landslide flipped the **House from D to 58R-42D** (POST 640) — a "liberal bloodbath" players found unrealistic, partly explained by the missing party-fatigue penalty (§1).
- **Design philosophy aired (authoritative):** Ted/vcczar — **hide the raw formulas/data in the app** ("fog of war"; show only which ideologies like a position, plus polls/maps), since visible math lets multiplayer collude/min-max; **historical state-biases will be a toggle** (historical-plausible vs. go-crazy modes) (POST 574, 593, 613).

## 4. Governance turn at modern scale (1960-62 → 1962-64) — full machinery exercised
This run drives the entire modern phase loop by hand. Confirms/extends many systems:

- **Factional Realignment (2.2):** faction-leader selection rules (must match faction ideology + an interest/lobby, have Leadership, not Lackey/Obscure; cascading relaxations if none qualify, POST 912); **kingmaker gifts** (+stats/expertise/traits to chosen pols, POST 924); **disgruntled-ideology flips** + **ideology shifts** (Prog→Lib etc., d-roll, flip-flopper penalty, POST 769, 771); **career-track seeding** (Private/Military/Judicial/Governing/Legislative/Admin/Backroom, POST 770-777); kingmaker→protégé chains.
- **Party-leader rolls (when applied):** 1st election: 75% +1 Command, 100% gain Leadership, 100% lose Obscure, **+3 in all primaries while leader**; re-election: 25% +1 Command, 20% each +1 Legis/+1 Gov/Kingmaker/Iron-Fist, **25% gain Unlikable + 25% Controversial**; **−1 party preference per term after the 3rd** (POST 933).
- **Congressional leadership:** factions nominate Speaker / Maj+Min Leader / Maj+Min Whip / Senate counterparts, gated by **Legislative thresholds** (Speaker open if no qualified pol; Maj Leader needs 4 legis, Whip 3, Sen Maj Leader 5, Sen Min Leader 4); kingmaker-weighted votes; winners earn points + trait/stat gains (lose Obscure, +1 legis, etc.) (POST 946-966). **Iron-Fist Speaker/Maj-Leader assigns the *minority's* committees too** (POST 964, 983).
- **Committees:** all 100 senators + all reps assigned to **Domestic / Economic / Foreign-Affairs / Judiciary** committees; **chairs gain a relevant expertise**; **Ranking Member now has in-game impact during committee votes** (POST 972-986). Majority/minority officers can't serve on committees.
- **Legislation (2.x):** each faction proposes bills sorted by committee; **committee vote → floor vote → "swaying"** (trait-based vote flips); some amendments **"cannot be packaged."** Modern era-keyed bill catalog observed: **Civil Rights Act of 1964, Medicare, Raise Social Security, High-Speed Rail, Federal Jobs Guarantee, McGovern-Fraser Commission, Increase NASA Funding, Increase Army/Navy, Ban Mandatory Union Dues, Abolish Poll Taxes Amd, DC EC Representation Amd, Line-Item-Veto Amd, Repeal 16th Amd, Cannabis Criminalization, "Allow government to treat Cuba as a client state"** (chunk-018 POST 1310-1311). Southern Democrats visibly vote **NAY** on the Civil Rights Act — region-not-party split is correctly modeled (chunk-018).
- **Cabinet confirmation:** committee vote → wide Senate vote → swaying; e.g. **Dean Rusk (Sec Navy) 90-0-10; Richard Helms (NSA→) 77-13-10** (chunk-016 POST 1150).
- **General/Random events** (meter-driven, A/B player choices, %-rolls): Major Diplomatic Conference (send self or Sec State; admin-graded → relations/credit up), Nationalist March, **Race Riots** (−1 dom-stab OR National Guard for −1 party-pref OR defer to governors), Coal-Mine Disasters, Major Earthquake (disaster-relief A/B), Outdated Weapons (−1 mil-prep), Ambitious Judge (Douglas may run for Pres 1964), Judicial Influence, "Among the People" (gain Likable), Regained Confidence (chunk-016 POST 1154; chunk-022 POST 1670).
- **Lingering Phase + Cabinet-impact-on-meters:** %-rolls move the 8 macro-meters (Revenue-Budget, Econ-Stab, Mil-Prep, Dom-Stab, Honest-Gov, Quality-of-Life, **Planet's Health**, + National Debt); cabinet officers (graded by relevant stat, "efficient" trait) net-adjust each meter (chunk-022 POST 1689-1692). Confirms the modern macro-state engine (#51 cluster) at scale.
- **Gov Actions:** ~all 50 governors take 1-2 actions (efficient = 2), %-roll pass/fail, awarding +100 lobby/ideology points and meter/industry/bias effects. Catalog: repeal/impose gov term-limits, allow/deactivate **state primaries**, **deactivate segregation**, gerrymander (+era-bias), build military academy, subsidize agriculture, drill natural gas, legalize gambling/lottery, ban teaching evolution, build political machine (chunk-022 POST 1694-1712).
- **SCOTUS:** legislated-variable court; justice **ideology drift** (Warren Lib→Prog, etc.); cases decided by **ideology vote + concurrence blocs + "shenanigans"** (Integrity blocks flips; Debater/Manipulative convert; Controversial+Manipulative+Iron-Fist president can pressure). Docket: **Mapp v. Ohio, Loving v. Virginia** both upheld 5-4 (chunk-022 POST 1713-1722). Confirms #52.
- **Deaths/Retirements + Special Appointments:** annual death/retirement rolls; **governors fill vacant Senate/House seats** or leave them vacant (chunk-022 POST 1662-1669).

## 5. End conditions / the nuclear-war "big red button" (corroborates #88)
- The GM frames the whole campaign around a **literal nuclear-war loss condition** — *"no one wins if that big red button gets hit"* (POST 1, intro) — alongside Vietnam, the Space Race, and "intense domestic turmoil."
- **★ But no nuclear-war / game-over event ever fired.** The run paused at ~1962-63 (the 1962-64 scripted block), well before any Cuban-Missile-Crisis analog. The "big red button" is **title + premise flavor**, NOT a mechanic observed in play. Cold-War content that *did* fire is **event/legislation-flavor on the generic engine** — "treat Cuba as a client state" (a bill), **Bay of Pigs** (used as a legislative/minority-flip lever, chunk-016 POST 1148), the **Nuclear Test Ban Treaty** (a scripted A/B "sign/don't-sign" event with relation/meter rolls, POST 1677-1688). This **directly corroborates #88 + #106**: there is no purpose-built nuclear/MAD subsystem and no observed meter-floor / nuclear game-end; the dramatic loss condition exists only as undocumented intent. (Compare `terror2000`, the only run that hit a meter-driven game-over.)

## 6. Multiplayer GM-workload signal (corroborates #114 / DH-36)
- A 10-human MP run with **two designers playing** — confirms the manual GM burden of running the modern phase loop at full 50-state, 10-faction, ~25-subphase scale is **enormous** (AP: "this lingering phase is gonna suck because I have to add all of the currently passed laws to the lingering phase sheet," POST 1688; "cabinet impacts on meters (kill me)," POST 1691).
- **★ The run died of that workload, not of a game-ending event:** original players moved to the 1916 test; AP's solo-resume attempt explicitly stalled on the manual labor — "going thru 50 states for all 10 factions might take awhile" (POST 1726). This is another data point that **the app's CPU/automation must own the manual sim** the forum GM does by hand — the same conclusion as `grass1772`/#114.
- Designer statement (corroborates the solo-first roadmap, #114): the visible-formula problem is multiplayer-specific ("the real game has multiplayer, so people will collude," POST 568); the fix is to **hide the math** in the app.
