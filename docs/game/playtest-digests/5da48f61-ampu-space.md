## ★ Geography-gated Gov action (sharpens #20)
The headline Gov-action datum. POST 2 (themiddlepolitical) adds: **"Propose to be official NASA launch
site state (Currently FL). Require leadership, and science or tech interest for governor. (Can default
to Florida)."** POST 3/4 (OrangeP47, themiddlepolitical) refine the GATE: launch sites must be
**equator-favored / extreme-Southern states or acquisitions near the equator** (real-world rationale:
closer to equator = cheaper launch; FL/French-Guiana precedent; coastal for crash-safety, POST 12).
Explicit mechanic (POST 4): **"You can make it easier for deep-south state gov's to have a successful
action, and harder for it to happen everywhere else."** → a **per-state success-probability modifier
on a Gov action** (a column of differential success odds keyed to state geography/identity).

This is a concrete instance of the **flat-vs-per-state Gov-action axis** (#20): today ALL gov actions
are state-agnostic (same odds for every state); this item REQUIRES per-state differential odds — the
same shape as the `completions` "double-success-chance column for certain Gov types" (Jim-Crow) under
#20. Same authoring pattern echoed by other proposed geo-gated Gov actions:
- **Build Space Elevator** — requires Panama/Colombia/Venezuela or good relations; must sit exactly
  on the equator (floating ocean platform; could tie sea-colonies) — territory/diplomacy-prereq'd
  (POST 3/4; spans Infrastructure+Diplomacy subtypes, cf. #248 multi-tag open Q).
- **Astronaut Training Facilities / vie for Mission Control (Houston)** — per-state Gov action,
  **first successful gov action gets the funding** (POST 9) — a one-winner race across states.
- **State space agency on NASA defunding** (POST 11) — if NASA defunded, govs spin up a state agency;
  **enough adjacent states with agencies can MERGE into a regional one** (e.g. northern/southern
  space agency) — a per-state Gov action with a state-adjacency aggregation rule (novel mechanic).
- **Military launch sites** noted in Southern CA / NV (Vandenberg AFB), broadening the gate beyond
  deep-South (POST 5/13).
