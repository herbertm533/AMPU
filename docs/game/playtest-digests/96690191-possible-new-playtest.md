## 2. Poll outcome ‚Üí the Gilded Age (1868) start

The poll resolves to the **Gilded Age, opening Jan 1869** (POST 21 GM "leaning toward
gilded age," POST 23/50/63 confirm):
- **Rationale (onboarding/mechanics):** GM picks 1868 because **Senators are
  governor-appointed** pre-17th-Amendment, so the GM only runs **House + Governor
  elections every 2 years** ‚Äî explicitly "easier for a newbie like me" (POST 5, 21,
  POST 23). A concrete reason a start-year choice changes the *election surface*.
- **Vote spread (non-binding):** Progressivism drew early votes (POST 2, 3, 13);
  Manifest Destiny / Gilded Age also favored (POST 6, 13). GM overrode toward Gilded
  Age for run-load reasons, not vote count.
- **Boot vision (POST 63):** start **January 1869, after Grant elected but before
  inauguration**; the **1868-elected Congress is in place**; players **draft new
  politicians per faction**, then elect leadership; GM will **ignore the historical
  1869 Speaker / President pro tem** to let players elect them; whoever controls
  **Grant** appoints cabinet/ambassadors; **military + SCOTUS pre-seated** (1 SCOTUS
  vacancy). This is a standard scenario-boot description (cf. #86), all hand-set by GM.
