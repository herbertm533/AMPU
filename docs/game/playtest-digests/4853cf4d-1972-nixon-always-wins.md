## What this thread is

The **second captured 1972-start** (players reference "the other 1972 thread"), and
the first to run a 1972 board as a *team multiplayer* game front-to-back through a full
half-term. Unlike the all-CPU/solo modern captures (`nuke`/`modern`/`terror2000`/`pop`),
this is **humans driving every faction** in the modern era — so it re-validates the
modern subsystem suite (#92/#110/#11/#45/#106/#34/#39) from a fresh start year and exposes
the **era-keyed-office and era-keyed-content** problems that a 1972 (vs 2000/2024) board
surfaces: offices and departments that don't exist yet in 1972, and bills flagged
inoperable as "Wrong Era." Watergate is discussed up front as a **percentage event gated
on the President's `Controversial` trait** — the canonical statement of that mechanic.

## Bugs / dataset / friction

- **★ DOE/DNI era-keyed-office data error:** the Jan-1972 start sheet listed a **Department
  of Energy** (real DOE = 1977) and a **"Director/Nat Intel"** office (real DNI = 2004) —
  both anachronistic for a 1972 board (POST 392-394, 274). The same class of bug as the
  2000-start's DNI/CIA mapping and as a 2024 board's DNI absence — **offices/departments
  must be era-keyed** so a start year only seeds the offices that exist then.
- **Janet Yellen −1 PV** flagged as harsh; vcczar (designer, POST 124): PV is just the sum
  of historical-career categories; Yellen is **Uncharismatic + Obscure + Lackey (−50)** so
  low (POST 121-125). Confirms **PV = office-weighted career tabulation** (`pv.ts`).
- Heavy **Google-Sheets tooling friction**: Master not pulling from faction tabs, faction-
  leader tab pulling wrong traits (Charisma false-positives), industry-leaders needing a
  bespoke auto-tally tab (POST 74-101, 264-267) — the recurring "build must own all upkeep"
  / "industry auto-tally is load-bearing" theme (#166, DH-36).
- "Admirals are slim pickings" in 1972 (POST 274) — dataset thinness in the modern military
  track (corroborates the modern-track-too-thin finding, #92/terror2000 POST 510-548).
- A 25%-roll convention: a 25% chance needs **≥75** on d100, not ≤25 (POST 162-164) — a GM
  arithmetic clarification, not an engine rule.
