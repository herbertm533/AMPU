# Playtest digests

One file per ingested forum thread, written by the `game-pm` agent during
`/ingest-playtest`. Each digest is a compact, citable summary of what that
thread revealed — eras/years covered, mechanics observed, game-master rulings,
house rules, and deltas vs. the current build — with references to the source
post numbers.

Digests are the **durable provenance** for the four living docs in the parent
folder. The bulky cleaned source text they summarize lives under
`docs/game/sources/` (gitignored, regenerable from the raw uploads).

_Empty until the first batch of forum files is ingested._
